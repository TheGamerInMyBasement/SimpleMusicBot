# MUSIC BOT — Self‑Hosted Discord Music Bot

A simple, fast, self‑hosted Discord music bot created by The_chosen_one (Discord) / TheGamerInMyBasement (GitHub).
No ads. No subscriptions. No limits. Just clean, cached audio playback.
Features

-  Local caching — Songs download once and replay instantly

-   Playlists — Automatically generates playlist.json

-   User controls — Block/unblock users from using music commands

-   Settings system — Stores configuration in settings.json

-   Request logging — Saves blocked requests in blocked_requests.json

-   Fast replay — Replays previously downloaded songs without re-downloading

## Installation

1.  Download or clone the repository

2.  Place the bot anywhere you want

3.  Create a .env file with:

TOKEN=your_bot_token_here
ADMIN_ID=your_discord_user_id

4. Install requirements.txt

5. install FFMPEG

6. This uses python 3.11 +
-----
## Running
To run the bot use the following command in your directory.

python bot.py
